# Lerumo Legacy Protection Service

Modern, professional security company website built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:5173` in your browser.

## 📦 Tech Stack

- **React 18** - UI library
- **Vite** - Fast build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Modern Design** - Clean, professional aesthetic

## 📁 Structure

```
src/
├── components/
│   ├── Navbar.jsx      - Navigation with mobile menu
│   ├── Hero.jsx        - Landing section
│   ├── Services.jsx    - Service cards
│   ├── About.jsx       - Company info & values
│   ├── Contact.jsx     - Contact form
│   └── Footer.jsx      - Footer
├── App.jsx             - Main component
├── main.jsx            - Entry point
└── index.css           - Global styles
```

## ✨ Features

✅ Fully responsive (mobile, tablet, desktop, 4K)
✅ Smooth animations with Framer Motion
✅ Dark theme with cyan accents
✅ Contact form with validation
✅ Mobile-friendly navigation
✅ Fast performance with Vite
✅ Modern glassmorphism design

## 🎨 Design

- **Primary**: Deep navy (#0F0F1E)
- **Accent**: Cyan (#00d9ff)
- **Glass effect**: Modern blur + transparency
- **Fonts**: Sora (display), Inter (body)

## 🌐 Deployment

Deploy to **Vercel**, **Netlify**, or **GitHub Pages**:

```bash
npm run build

```
